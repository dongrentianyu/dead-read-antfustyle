在cardView.astro中有下面这样一段代码，这段代码就定义了如何shorts 这个collectionType，换言之，同样我们也可以在组件中增加和定义新的代码，并保证，来增加更多的筛选。

注意下面的筛选中需要保持

```
if (collectionType === ('shorts' as CollectionKey)) {
  const posts = await getCollection('blog', ({ data }) => {
    return data.description.includes('test')
  })
  dataForGrid = await getShortsFromBlog(posts)
}
```
